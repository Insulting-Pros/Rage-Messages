# Rage Messages

Prints scrake, fleshpound rage info to player chat.

## Installation

```cpp
RgMsgMut.RgMsgMut
```

## Building and Dependancies

At the moment of 2021.03.27 there are no dependencies.

**EditPackages**

```cpp
EditPackages=RgMsgMut
```

## Config Files

[RgMsgMut.ini](Configs/RgMsgMut.ini 'main config')

* Rage message.
* Scrake / fp switches.
* Color settings.
